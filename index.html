Numerical Schrödinger solver

This discretises, live in your browser, the radial l=0 Coulomb equation −½u″ − (Z/r)u = E u and diagonalises it (finite-difference + QL). This is a real eigensolve, not the formula — compare its energies to the closed form, and see the actual wavefunctions.

Pick an element and solve. (Hydrogen is the exact benchmark.)

Quantum circuit (1–3 qubits)

Choose how many qubits, build a circuit from gates, and measure. Your browser computes the full state vector, each qubit's Bloch vector & expectation values ⟨X⟩⟨Y⟩⟨Z⟩, entanglement, and a sampled measurement histogram — plus all the qubit equations with live values.

H₂ molecule — dissociation curve

This computes H₂ from STO-3G integrals and a 2-configuration full-CI in the σ_g/σ_u molecular orbitals (it diagonalises a real molecular Hamiltonian). The ground state is a bonding/antibonding superposition — an effective qubit.

Solve to plot the famous H₂ energy curve E(R).

Wave packet — time-dependent Schrödinger

Split-step Fourier evolution of |ψ(x,t)|² (ℏ=m=1). Free packet drifts at k₀ and spreads; harmonic oscillates; barrier shows tunnelling/reflection; soliton holds its shape.

Evolve to animate the wave packet.

Quantum error correction — the below-threshold story

The gateway to fault-tolerant quantum computing: a logical qubit encoded in many physical qubits gets better as you add qubits — but only if the physical error rate is below a threshold. This recreates that result classically. The repetition (bit-flip) code is a real, exactly-simulable code (logical error computed exactly and Monte-Carlo-checked); the surface code uses the standard sub-threshold scaling model.

Run with p below 0.5 (repetition) and watch larger distances drive the logical error down.

🧪 The physics is real

This isn't a set of pre-rendered animations — a single in-browser engine solves the actual equations, live:

  • Schrödinger — finite-difference radial eigensolve (QL algorithm, 500-point grid), benchmarked against the exact hydrogen levels.
  • Quantum circuits — a full complex state-vector simulator: gates, Bloch vectors, reduced density matrices, entanglement, sampled measurement.
  • H₂ molecule — STO-3G Gaussian integrals + 2-configuration full-CI → the dissociation curve E(R) and an effective qubit Hamiltonian.
  • Wave packets — the time-dependent Schrödinger equation by split-step Fourier (free / harmonic / barrier tunnelling / soliton).
  • Error correction — repetition-code logical error (exact + Monte-Carlo) and the surface-code sub-threshold story.
  • Beams & bonding — photoelectric effect, Compton & Rutherford scattering, and ionic/covalent/metallic bond prediction.

Validated against known values — hydrogen −13.6 eV, Bell-state entanglement, H₂ ≈ 0.74 Å / −1.14 Ha, and more.

⚙️ Under the hood — powered by Akka

The full lab is engineered on distributed Akka actor clusters (Java) — I build and run the whole thing on Akka. The physics engine, the 119-element periodic-table server, the knowledge engine and the Quanta AI assistant all run as cooperating Akka actors across the cluster.

For this public, static build, that physics was faithfully re-implemented in pure browser JavaScript and wired behind a tiny fetch shim — so the demo you're using needs no server and runs 100% on your device.
